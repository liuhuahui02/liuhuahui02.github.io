初识ES6

  • ECMAScript 6.0（以下简称 ES6）是 JavaScript 语言的下一代标准，已经在 2015 年 6 月正式发布了。
  • ECMAScript 是 Javascript 语言规范

let 和 const

  • let 块级声明变量，只在它所在的代码块内有效，且不存在变量提升(不允许提前解析)，不允许重复声明
  • const 声明变量 是 不可变的常量，必须初始化赋值

模板字符串

  • 字符串的扩展
    • includes() 判断字符串中是否包含指定的字符，参一：匹配的字符串，参二：从第几个开始
    • startsWith() 判断字符串是否以特定的字符开始
    • endsWith() 判断字符串是否以特定的字符结束

数组的扩展

  • 扩展运算符 （…） 展开数组

模块 （export 和 import）

  • export 用于规定模块的对外接口，向外抛出变量或方法
  • import 用于输入其他模块提供的功能， 即引用别的文件抛出的变量方法
