初识nunjucks模板引擎

分享

文档API: https://nunjucks.bootcss.com/

JavaScript 专用的功能丰富、强大的模板引擎

Node端使用

$ npm install nunjucks

extends模板继承 + block区块 + include引入其他模板 实现网页布局，（即头部，侧边导航栏，底部 提炼出 相同部分模板）

目录： /views
footer.html 底部
header.html 头部
sidebar.html 侧边栏
index.html 首页
layout.html 布局页 （公共页面）
